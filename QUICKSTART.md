# Quickstart — CCE Audio Studio

## Windows (fast path)
1) Run `_BAT\1_setup.bat`
2) Run `_BAT\2_run.bat`
3) Run `_BAT\3_open_browser.bat`

## First test
- Open `/ready` and confirm **Health = status: ok**.
- Create a new project.
- Click **Generate Pilot Pack**.
- Click **Approve**.
- Click **Export ZIP**.

## Tests
- `_BAT\6_run_tests.bat`
- Or: `python tools/run_full_tests.py`
